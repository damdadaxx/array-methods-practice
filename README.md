```
array-methods/
```

## 🧭 학습 가이드

### 🔰 비기너 / 미숙련자

> **1번 ~ 7번 기본 문제**부터 차근차근 풀어보세요.

- 배열 메서드를 처음 접하거나 아직 익숙하지 않은 분을 위한 문제입니다.
- 각 메서드의 기본 문법과 동작 원리를 이해하는 데 집중하세요.
- 순서대로 풀면 자연스럽게 난이도가 올라갑니다.
- 추천 학습 순서: `forEach` → `filter` → `map` → `reduce`

### 🚀 기본 사용이 가능한 분

> **8번 ~ 10번 응용 문제**로 바로 넘어가세요.

- 이미 기본적인 배열 메서드 사용이 가능한 분을 위한 문제입니다.
- 객체 배열 다루기, 복합 조건, 데이터 변환 등 실전에서 자주 마주하는 패턴을 연습합니다.
- 기본 문제가 너무 쉽게 느껴진다면 바로 응용 문제에 도전해 보세요.

## 💡 문제 풀이 방법

1. 각 메서드 폴더에서 `1.js`부터 열어보세요.
2. 파일 안의 `// TODO:` 주석을 읽고 지시에 따라 코드를 작성하세요.
3. 터미널에서 `node forEach/1.js` 처럼 실행하여 결과를 확인하세요.
4. 정답이 궁금하면 `solution/` 폴더에서 같은 번호의 파일을 확인하세요.

```bash
# 문제 실행 예시
node forEach/1.js

# 정답 확인 예시
node solution/forEach/1.js
```
